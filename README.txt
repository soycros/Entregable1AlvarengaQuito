Simulador - Estilo entrega - Listado de tareas con API incluido con CDN
El mismo tiene como principal tarea el arreglo y organizacion de las tareas por medio de una lista simple y facil de usar. Esta con colores de contraste.
El mismo cuenta con lista dinamica que se puede usar para el marcado de tareas de formas progresiva y con una interfaz colorida y util en tareas de vaciado, llenado y de mas.
Funcionalidades principales

-  Añadir tareas dinámicamente
-  Marcar tareas como completadas
-  Guardar la lista de tareas en `localStorage`
-  Cargar tareas desde una API externa (`jsonplaceholder`)
-  Visualización y actualización del DOM en tiempo real
-  Uso de alertas visuales
-  Notificaciones sutiles
-  Diseño oscuro elegante y responsivo
